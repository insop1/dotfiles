# dotfiles

Fedora 44 + Hyprland.

# important

The waybar config uses [insop-media](https://github.com/Insop1/insop-media), you can either remove the custom module or build it.

```bash
git clone https://github.com/Insop1/insop-media ~/.config/waybar/scripts/insop-media
cd ~/.config/waybar/scripts/insop-media
cargo build --release
```
